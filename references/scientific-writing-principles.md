Section-level structure is in the section-specific guides; manuscript-level workflow is in `references/writing-process.md`. ## The Six Pillars

Effective scientific writing rests on six pillars. Apply them in revision passes:

1. **Clarity.** The reader should not have to re-read.
2. **Conciseness.** Every word earns its place.
3. **Accuracy.** Numbers, terms, and claims match the evidence.
4. **Objectivity.** Findings stated without rhetorical inflation.
5. **Consistency.** One term per concept; one notation throughout.
6. **Logical organization.** Each paragraph and section flows from the previous.

## Conciseness — Wordy → Concise

The single most impactful revision move is cutting fillers. Every removed word increases the density of meaning.

| Wordy | Concise |
| --- | --- |
| `due to the fact that` | `because` |
| `in order to` | `to` |
| `it is important to note that` | (delete; just say it) |
| `a total of 50 participants` | `50 participants` |
| `completely eliminated` | `eliminated` |
| `has been shown to be` | `is` |
| `in the event that` | `if` |
| `at the present time` | `now` / `currently` |
| `conduct an investigation into` | `investigate` |
| `give consideration to` | `consider` |
| `make a decision` | `decide` |
| `perform an analysis of` | `analyze` |
| `make an assessment of` | `assess` |
| `provide information about` | `inform` |
| `in spite of the fact that` | `although` |
| `with regard to` | `regarding` / `for` |
| `during the course of the study` | `during the study` |
| `the majority of` | `most` |
| `a small number of` | `a few` |
| `at this point in time` | `now` |
| `for the purpose of` | `to` / `for` |
| `it is possible that` | `may` / `might` |
| `subsequent to` | `after` |
| `prior to` | `before` |
| `in close proximity to` | `near` |
| `Measurements were performed to evaluate the levels of CEA in the serum.` | `We measured serum CEA.` |
| `No instances of either postoperative death or major complications occurred.` | `There were no postoperative deaths or major complications.` |
| `The procedure was performed with the use of` | `We used` |

### Avoid throat-clearing phrases

```
Wordy:    It is interesting to note that the results of our study demonstrate that...
Concise:  Our results show that...
```

### Strong verbs over noun + verb

Nominalization (turning a verb into a noun) makes prose heavy and indirect. Convert back to the verb.

| Nominalized | Direct |
| --- | --- |
| `make a decision` | `decide` |
| `make an assumption` | `assume` |
| `perform an investigation` | `investigate` |
| `conduct an examination` | `examine` |
| `achieve a reduction` | `reduce` |
| `provide an explanation` | `explain` |
| `give consideration to` | `consider` |
| `effect a change` | `change` |

### Cut unnecessary intensifiers

`Very`, `quite`, `rather`, `extremely`, `highly` rarely add information in scientific writing. The data carry the magnitude.

```
Wordy:    The results were very significant.
Concise:  The results were significant. (the p value conveys the degree)
```

### Reasonably long sentences are fine

Length itself is not the problem; redundancy is. A long sentence packed with necessary technical detail is acceptable:

> We analysed blood samples using liquid chromatography-tandem mass spectrometry (LC-MS/MS) on a Waters Acquity UPLC system coupled to a Xevo TQ-S mass spectrometer (Waters, Milford, MA, USA).

The diagnostic question on every revision pass: **can any word be removed without losing meaning?** If yes, remove it.

## Hedging — Calibration of Certainty

Scientific writing requires calibrated confidence. Both extremes harm credibility.

### Excessive hedging

```
It could perhaps be possible that the intervention might possibly have some effect on
symptoms under certain conditions.
```

Reads as evasive; reviewers conclude that the authors do not believe their own data.

### Insufficient hedging

```
The intervention cures depression.
```

Reads as overclaim; reviewers reject for overreach.

### Calibrated hedging

```
The intervention significantly reduced depressive symptoms in this sample, suggesting
that it may be effective for mild to moderate depression.
```

### Hedging vocabulary, by purpose

| Purpose | Words |
| --- | --- |
| Interpretation of association | `suggests`, `indicates`, `is consistent with`, `supports` |
| Causal language for non-RCT | `was associated with` (not `caused`, `reduced`, `improved`) |
| Causal language for well-conducted RCT | `reduced`, `increased`, `did not change` (cautious `caused`) |
| Possibility | `may`, `might`, `could` |
| Surface observation | `appears to`, `seems to` |
| Degree of certainty | `likely`, `probably`, `possibly` |

Match the strength of the verb to the design (see also `references/study-types.md`).

## Anthropomorphism — Don't Let the Paper Speak

Scientific objects (studies, data, tables, papers) cannot want, hope, or claim. Only people can.

| Anthropomorphic | Scientific |
| --- | --- |
| `The study wanted to examine ...` | `We aimed to examine ...` / `The study examined ...` |
| `The data suggest they want ...` | `The data suggest that ...` |
| `This paper will prove ...` | `This paper demonstrates ...` |
| `Table 1 tells us that ...` | `Table 1 shows that ...` |
| `The literature claims ...` | `Authors of recent reviews argue ...` |
| `The evidence demonstrates that ...` | `Smith and Jones demonstrated that ...` |

A useful rule: never let evidence, results, or papers be the subject of a verb that requires intent.

## Voice — Active vs. Passive

Active voice is generally clearer and shorter. Use it where it clarifies who did what.

```
Passive:  The samples were analyzed by the researchers.
Active:   We analyzed the samples.
```

Passive voice is acceptable — and sometimes preferred — when the **action matters more than the actor**:

```
Acceptable passive: Blood samples were collected at baseline and after 6 weeks.
Acceptable passive: Outcomes were adjudicated by a committee blinded to assignment.
```

Many medical journals now actively prefer active voice with `we`. Do not refrain from `we`.

## Tense — A Section-by-Section Reference

Tense varies by section and even by paragraph type. Default rules:

| Section | Sentence type | Tense | Example |
| --- | --- | --- | --- |
| Abstract Background | Established knowledge | Present | `Stroke is a leading cause of death.` |
| Abstract Background | Prior studies | Past or present perfect | `Trials have shown that ...` / `Smith et al. reported that ...` |
| Abstract Methods | Study procedures | Past | `We randomly assigned 4,250 adults ...` |
| Abstract Results | Findings | Past | `The primary outcome occurred in 21.7% ...` |
| Abstract Conclusions | Take-home | Present | `Dapagliflozin reduces hospitalization ...` |
| Introduction — burden | Established | Present | `Sepsis affects 49 million people annually.` |
| Introduction — prior studies | Citations | Past | `In 2019, Smith et al. found ...` |
| Introduction — gap | Knowledge state | Present | `Whether early treatment improves survival is unclear.` |
| Introduction — present study | Aim | Past or present | `We investigated ...` / `This study examines ...` |
| Methods | Study actions | Past | `We enrolled adults aged 18 or older.` |
| Methods | Standard procedures | Present | `qRT-PCR amplifies cDNA ...` |
| Results | Findings | Past | `Mean age was 72 years.` |
| Discussion — own findings | Recap | Past | `In this trial, dapagliflozin reduced ...` |
| Discussion — interpretation | Inference | Present | `This finding suggests ...` |
| Discussion — established knowledge | Background | Present | `SGLT2 inhibition lowers glucose ...` |
| Discussion — future research | Recommendation | Future or modal | `Future trials should compare ...` / `Larger studies will be required ...` |
| Conclusion | Take-home | Present | `Dapagliflozin reduces hospitalization in HFpEF.` |

If a paragraph mixes tenses, check whether each sentence really belongs to its section's purpose.

## Abbreviation Discipline

Excessive abbreviation slows reading. Apply two rules:

1. **Define each abbreviation at first use** in the abstract and again at first use in the body text. The abstract's definitions do not carry over.
2. **Only abbreviate terms used three or more times** in the same section. If you need it twice, write it out.

Avoid abbreviation in titles and section headings. Limit new abbreviations to roughly six per abstract and ten per body. Standard abbreviations (DNA, RNA, HIV, MRI, CT, ECG, IV, PO) do not need definition.

```
Poor:    We measured Brain-Derived Neurotrophic Factor (BDNF) at baseline.
         BDNF levels were elevated.   (used twice; abbreviation unnecessary)
Better:  We measured brain-derived neurotrophic factor at baseline. Levels were elevated.
```

## Numbers and Units

### Numerals vs. words

| Use numerals | Use words |
| --- | --- |
| Numbers ≥ 10 | Numbers < 10 (without units) |
| Numbers with units (`5 mg`, `3 mL`) | Numbers starting a sentence (or restructure) |
| Statistical values (`p = 0.03`, `t = 2.14`) | Approximate quantities (`one in three patients`) |
| Ages, dates, times | |
| Scores and scales | |
| Percentages (`15%`) | |

```
Five participants withdrew.        (or: There were 5 withdrawals.)
We tested 15 samples at 3 time points.
Mean age was 45 years.
```

Sentence rule: never start a sentence with a numeral. Either spell it out or restructure.

### Units

1. Insert a non-breaking space between number and unit (`5 mg`, `60 kg`, `12 weeks`). One exception: percent has no space (`15%`).
2. No period after units (`mg` not `mg.`).
3. SI units by default; convert non-SI conventions only when the field expects them (e.g., `mm Hg` for blood pressure, `mmol/L` for glucose in some regions).
4. Consistent decimal precision per metric across the manuscript.
5. Use commas for thousands in body text (`12,500`); some journals use spaces (`12 500`) — check the Instructions.

### Ranges

Use an en-dash (`–`) or `to` for ranges; include the unit only after the second number.

```
15-20 mg          (en-dash; preferred in many journals)
15 to 20 mg       (alternative; required by some journals — consistent with the no-em-dash rule)
NOT 15 mg-20 mg
```

## Common Clarity Problems

### Ambiguous pronouns

```
Poor:  Group A received the drug and Group B received placebo. They showed improvement.
       (Who is "they"?)
Better: Group A received the drug and Group B received placebo. The drug-treated group
        showed improvement.
```

### Misplaced modifiers

```
Poor:   We measured blood pressure in patients using an automated monitor.
        (Are the patients using the monitor, or are we?)
Better: Using an automated monitor, we measured blood pressure in patients.
```

### Dangling modifiers

```
Poor:   After incubating for 2 hours, we measured absorbance.
        (Suggests "we" were incubated.)
Better: After incubating samples for 2 hours, we measured absorbance.
```

### Unclear referent for "this"

```
Poor:   The increase in expression was accompanied by decreased proliferation,
        which was unexpected.
Better: The increase in expression was accompanied by decreased proliferation;
        this inverse relationship was unexpected.
```

A simple discipline: never start a sentence with `This` followed by a verb. Always pair `This` with a noun (`this finding`, `this association`, `this design choice`).

## Commonly Confused Words

| Often misused | Correct usage |
| --- | --- |
| **affect / effect** | `affect` (verb): influence; `effect` (noun): result; `effect` (verb): bring about |
| **among / between** | `among`: three or more; `between`: two |
| **continual / continuous** | `continual`: repeated with breaks; `continuous`: uninterrupted |
| **data is / data are** | `data are` (plural in formal scientific writing); `datum` (singular). Some journals now accept `data is`; check style. |
| **fewer / less** | `fewer`: countable; `less`: continuous quantities |
| **i.e. / e.g.** | `i.e.` = that is (restatement); `e.g.` = for example (examples). Both followed by a comma in US style. |
| **imply / infer** | `imply`: suggest (the speaker); `infer`: deduce (the listener) |
| **parameter / variable** | `parameter`: population value; `variable`: measured characteristic. Often misused — `parameter` is rarely the right word for "variable" or "metric". |
| **principal / principle** | `principal`: main; `principle`: rule or concept |
| **significant** | Reserve for **statistical significance**; do not use as a synonym for "important" |
| **that / which** | `that` introduces a restrictive clause (no comma); `which` introduces a non-restrictive clause (with comma) |
| **comprise / compose** | The whole **comprises** the parts; the parts **compose** the whole. Avoid `is comprised of`. |
| **regimen / regime** | `regimen`: a treatment schedule; `regime`: a system of government |
| **dose / dosage** | `dose`: the specific amount given at one time; `dosage`: the schedule of doses |
| **aetiology / etiology** | British vs. American spelling — be consistent |

## Tone — Avoid Subjectivity

```
Subjective:  As expected, our superior method performed better.
Objective:   Method A showed higher accuracy than Method B (87% vs. 76%, p = 0.02).
```

```
Subjective:  The results were disappointing and concerning.
Objective:   The intervention did not significantly reduce symptoms (p = 0.42).
```

Acknowledge inconsistent evidence directly:

```
Our findings contrast with Smith et al. (2022), who reported no significant effect.
This discrepancy may result from differences in intervention intensity or population
characteristics.
```

Distinguish observation from interpretation:

```
Observation:    Mean blood pressure decreased from 145 to 132 mm Hg (p = 0.003).
Interpretation: This suggests that the intervention effectively lowers blood pressure.
```

## Accuracy — Calibrate Numbers

Match numerical precision to measurement precision:

```
Inappropriate (false precision): Mean age was 45.237 years.
Appropriate:                     Mean age was 45.2 years.
```

Match statistical claims to the test result:

```
Incorrect: The correlation was highly significant (p = 0.03).
Correct:   The correlation was statistically significant (p = 0.03).
           ("highly significant" is reserved for p < 0.001 if used at all;
           reporting the exact p value is preferable.)
```

Verify on every revision pass:

1. Numbers in body text match tables and figures.
2. `n` values sum correctly.
3. Percentages match the numerator/denominator they claim.
4. Effect estimates match their CIs.

## Logical Organization — The "Red Thread"

Every section, paragraph, and sentence chain should advance one main argument. Within a paragraph:

1. **Topic sentence** — one main idea.
2. **Supporting sentences** — evidence, explanation, exception.
3. **Concluding or transition sentence** — closes the idea or bridges to the next paragraph.

Across sections:

1. The question raised in the Introduction is answered in the Results.
2. Methods choices reflect the design implied by the question.
3. The Discussion interprets the Results without introducing new data.

### Signposting

```
First, we examined ...
Next, we investigated ...
Finally, we assessed ...
```

### Parallelism

```
Not parallel: Aims were to (1) measure blood pressure, (2) assessment of cognitive
              function, and (3) we wanted to evaluate mood.
Parallel:     Aims were to (1) measure blood pressure, (2) assess cognitive function,
              and (3) evaluate mood.
```

## Punctuation — Quick Reference

The em-dash rule (no em-dashes in body sentences) is in `references/manuscript-conventions.md`. Other quick rules:

### Comma placement

```
Wrong: The participants in the intervention group, showed improvement.
       (no comma between subject and verb)
Right: The participants in the intervention group showed improvement.

Wrong: We measured blood pressure, and recorded symptoms.
       (no comma before `and` joining a compound predicate)
Right: We measured blood pressure and recorded symptoms.

Right: We measured blood pressure, and we recorded symptoms.
       (independent clause; comma is appropriate)
```

### Subject-verb agreement

```
Wrong: The group of participants were heterogeneous.
       (subject is `group`, singular)
Right: The group of participants was heterogeneous.

Right: The participants were heterogeneous.   (plural subject)
```

### Pronoun agreement

```
Acceptable: Each participant completed his or her questionnaire.
Better:     Participants completed their questionnaires.
```

## Paragraph Length and Coherence

Target 100–200 words per paragraph (3 to 7 sentences). Too short suggests an underdeveloped idea or one that should be merged with a neighbour; too long suggests two ideas that should be split.

Coherence devices within a paragraph:

1. **Topic sentence** that names the idea.
2. **Repetition of key terms** (do not switch to synonyms for "variety").
3. **Transitional phrases**: `first / second / finally`, `furthermore / moreover`, `however / nevertheless`, `therefore / thus / consequently`, `for example / specifically`.

## Tools — Use, but Don't Trust

Grammar and style checkers (Grammarly, ProWritingAid, Hemingway Editor, LanguageTool) are useful as a first pass but do not understand scientific conventions: they flag passive voice in Methods (which is often fine), suggest hedging where calibration is needed, and miss subject-matter inconsistencies. Apply them as a starting point, then review with the rules in this file.

## Sentence-Level Revision Checklist

For each paragraph during the second revision pass:

1. One main idea, stated in the topic sentence?
2. Each sentence ≤ 25–30 words?
3. Active voice where it clarifies; passive only when the action matters more than the actor?
4. Tense matches the section?
5. Every abbreviation defined at first use; only used three or more times?
6. Numbers formatted correctly (numerals, units, ranges, no sentence-starting numerals)?
7. No anthropomorphism (`the study aimed`, `this paper proves`)?
8. Hedging calibrated to the evidence (`suggests` for observational, `reduced` for RCT)?
9. Pronouns have unambiguous antecedents?
10. `This` always paired with a noun?
11. Confused-words pairs (`affect/effect`, `which/that`, `data is/are`) checked?
12. No em-dashes in body sentences (see `references/manuscript-conventions.md`)?
