# Methods Writing — Common Issues (Reference Note)

Use this troubleshooting checklist after drafting Methods.

## Top issues

1. **Missing reporting-checklist items.** Map the section against CONSORT / STROBE / PRISMA / STARD before finalizing. The most commonly missed items are sample size justification, allocation concealment (RCT), risk-of-bias assessment (review), and handling of indeterminate results (diagnostic).
2. **Procedure-only writing.** A Methods subsection lists what was done but never says why. Add a rationale sentence at the start of each subsection.
3. **No safeguard.** A subsection describes what was done but does not explain how bias was controlled. Add blinding, prespecification, validation, or sensitivity analysis.
4. **Inconsistent terminology.** The exposure or outcome is named differently across subsections. Use exactly one term per concept and define it once.
5. **Vague statistics.** "We adjusted for confounders." Name the variables, the model, and the software with version.
6. **Surrogate outcomes treated as primary without justification.** Either justify the surrogate (mechanistic and empirical evidence) or downgrade.
7. **Missing data hand-waved.** State the mechanism assumed (MCAR / MAR / MNAR) and the handling method.
8. **Subgroups undeclared.** All subgroups were prespecified — except the ones that were post-hoc, which are now reported without that label.
9. **Citation order broken.** Citation 23 appears before citation 18. Renumber after every revision.
10. **Em-dashes in body sentences.** Replace `—` with commas, semicolons, or parentheses.

## Quick fix checklist

1. Each subsection has rationale + procedure + safeguard.
2. Each statistical method names a model, covariates, software, version.
3. Each measurement names an instrument, units, and timing.
4. Sample size has assumptions, formula, and inflation.
5. Bias-control mechanisms are explicit (blinding, prespecification, validation).
6. Reporting-checklist items each have a paragraph or page number.
7. Citations and tables/figures are in order of first appearance.
8. No em-dashes.
