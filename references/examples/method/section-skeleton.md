# Methods Section Skeleton

## RCT (CONSORT)

```
Methods
% Trial design (parallel / crossover / cluster, allocation ratio)
% Participants (setting, eligibility, recruitment, consent)
% Interventions (active and comparator: dose, schedule, duration, adherence monitoring)
% Outcomes (primary, secondary, adverse events, ascertainment, blinded adjudication)
% Sample size (assumptions, formula, inflation for dropout)
% Randomization and blinding (sequence, concealment, blinding of who)
% Statistical analysis (population, model, missing data, multiplicity, sensitivity, software)
% Ethics, registration, role of funder
```

## Observational study (STROBE)

```
Methods
% Study design (cohort / case-control / cross-sectional)
% Setting and period
% Participants (eligibility, sources, methods of selection, follow-up)
% Variables (exposure, outcome, confounders, effect modifiers; operational definitions)
% Data sources / measurement (and any validation)
% Bias (selection, information, confounding)
% Study size
% Quantitative variables (handling, categorization)
% Statistical methods (model, confounder adjustment, missing data, sensitivity, subgroups)
% Ethics
```

## Diagnostic accuracy study (STARD)

```
Methods
% Study design (cross-sectional, cohort, case-control)
% Participants (eligibility, sampling, consecutive vs. selected)
% Index test (description, who performed, blinding, threshold)
% Reference standard (description, blinding, time interval to index test)
% Test methods and analysis (handling of indeterminate results)
% Sample size
% Statistical methods (sensitivity, specificity, predictive values, AUC, 95% CIs)
% Ethics
```

## Systematic review / meta-analysis (PRISMA)

```
Methods
% Protocol and registration (PROSPERO)
% Eligibility criteria (PICO + designs + dates)
% Information sources and search dates
% Search strategy (full strategy in supplement; example in main text)
% Selection process (number of reviewers, blinding, software)
% Data collection process and items
% Risk-of-bias assessment (RoB 2 / ROBINS-I / QUADAS-2 / Newcastle-Ottawa)
% Effect measures
% Synthesis methods (qualitative summary, pooled estimate, model, heterogeneity, subgroup, sensitivity)
% Reporting bias assessment (funnel plot, statistical tests)
% Certainty of evidence (GRADE)
```
