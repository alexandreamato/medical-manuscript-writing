# Knowledge Gap Version 3 — Novel Clinical Question

`Version 3: For a novel question (a newly recognized syndrome, a new diagnostic, a recent regulatory change), define the question directly and decompose the difficulty into concrete points.`

## Skeleton

```
% State the goal and that the question is challenging for N reasons
In this study, our aim is to [AIM]. This question is challenging for three reasons.

% Point 1
First, ...

% Point 2
Second, ...

% Point 3
Finally, ...
```

## Worked example (illustrative; fictional)

```
In this study, our aim is to estimate the long-term cardiopulmonary sequelae of
post-acute COVID-19 in adults discharged from intensive care during 2020 to 2022.
This question is challenging for three reasons.

First, the syndrome is recently recognized and lacks a consensus operational
definition; symptom-based criteria, imaging-based criteria, and exercise-tolerance
criteria each capture different patient subgroups, and reported prevalence
varies five-fold across studies.

Second, exposure to SARS-CoV-2 was widespread during this period, so a contemporaneous
contemporaneous unexposed comparator is rare; cohort-style comparisons must rely on
historical or non-infected controls with imperfect matching on baseline cardiopulmonary
status.

Finally, follow-up is constrained by attrition: many ICU survivors are lost to
follow-up because of relocation, death from competing causes, or non-attendance, and
attrition is plausibly informative because patients with severe sequelae may be more or
less likely to remain engaged.
```

## Notes

1. Use `First / Second / Finally` to keep the points visually independent.
2. Each point states the observable difficulty and the underlying methodological or clinical reason.
3. Use this version sparingly and only when the question genuinely is new; otherwise reviewers will read the framing as inflation of novelty.
