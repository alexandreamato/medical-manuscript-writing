# Design Version 4 — Observation-Driven Contribution

`Use when the contribution comes from one striking observation or pilot finding that the present study confirms in a definitive design.`

## Skeleton

```
% State the key methodological or biological insight
Our key insight is [INSIGHT].

% State the observation that motivated it
We observed that [PILOT SIGNAL / REGISTRY TREND / MECHANISTIC FINDING] suggested [HYPOTHESIS].

% State the present study that tests it
Considering this, we conducted [DESIGN] to test whether [HYPOTHESIS] in [POPULATION].

% State the expected practical impact
If confirmed, [IMPACT].
```

## Worked example (illustrative; fictional)

```
Our key insight is that, in patients with HER2-low breast cancer, response to
trastuzumab deruxtecan extends to tumours previously classified as HER2-negative
under the binary 0 / 1+ vs. 2+ / 3+ framework.

We observed in a single-centre case series of 42 patients that pretreatment HER2 1+
tumours showed objective responses comparable to HER2 2+ tumours under conventional
scoring.

Considering this, we conducted an open-label, single-arm phase 2 trial in 180 patients
with previously treated HER2-low metastatic breast cancer to estimate the objective
response rate and durability of response across HER2 immunohistochemistry strata.

If confirmed, this finding would support reclassification of the HER2-low population
as a treatment-eligible group and would broaden the indication of trastuzumab
deruxtecan beyond current labelling.
```

## Notes

1. The observation must be reproducible and pre-existing; do not invent a pilot to motivate the design after the fact.
2. The "if confirmed" sentence frames the practical stakes for clinicians and reviewers.
3. Phase 2 single-arm designs do not establish causality; the writing should respect that.
